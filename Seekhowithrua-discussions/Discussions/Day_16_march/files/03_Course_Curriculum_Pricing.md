# Course Curriculum, Pricing & Hybrid Format
*Full Stack AI Builder — 18 Weeks*

---

## Course Philosophy

In the AI era, breadth beats depth for beginners. Claude, ChatGPT, Kimi — these tools write 80% of code. What AI cannot do: understand the problem, pick the architecture, debug weird errors, direct the tools. A student who understands basics of everything becomes a reliable builder. That is what this course creates.

**Course name: Full Stack AI Builder**
*Web + AI + Data + IoT + Gaming + DevOps — All in One*

## AI Tools Integration (Module 3 highlight)

Teach students to use ALL tools knowing WHICH to use WHEN:

- **Claude** — reasoning, writing code logic, building complete apps
- **ChatGPT** — quick answers, brainstorming, debugging
- **Kimi / Gemini** — long documents, research, large context
- **Cursor / v0** — AI-powered code editors for faster building
- **GitHub Copilot** — inline suggestions while coding

This "AI Tool Mastery" session alone is worth the entire course fee for students.

## Hybrid Class Format — 3 Days Per Week

| Day | Format | Duration | Content |
|---|---|---|---|
| Monday | Online (Google Meet) | 90 min | New concept taught — recorded automatically |
| Wednesday | Online (Google Meet) | 60 min | Doubt clearing + code review + assignment |
| Saturday | In-person (lab/coworking) | 3 hours | Hands-on project building + team work |

### Why Hybrid Beats Both Pure Options

| Format | Completion rate | Problem |
|---|---|---|
| Pure online | 8–15% | Students disappear |
| Pure offline | 60–70% | Limited to one city |
| Your hybrid | 65–75% | Best of both |

### Saturday Venue Options — Noida
1. **Partner college computer lab** — Free. Offer co-branded certificate + free seminar per semester in return.
2. **Co-working space** — Awfis Noida, 91springboard, IndiQube — Rs 300–600/day.
3. **Your home** — 10–15 students max. Most personal and powerful.

### Running Two Parallel Batches
- Batch A: Monday morning + Wednesday morning + Saturday morning
- Batch B: Tuesday evening + Thursday evening + Saturday evening
- 2 batches x 20 students x Rs 5,999 = Rs 2.4 lakhs per 18-week cycle
- 2 cycles per year = Rs 4.8 lakhs minimum

### Recording System
Record every Monday and Wednesday session using OBS or Google Meet auto-record. Upload to private YouTube playlist same evening. After 3 batches you have 100+ recorded sessions — this becomes SeekhoWithRua's automated course library.

### Weekly Rhythm for Student
- Monday: Learn new concept (90 min live)
- Tue–Wed: Practice independently + small assignment
- Wednesday evening: Doubt clearing + code review (60 min)
- Thu–Fri: Complete assignment
- Saturday: In-person lab — build project piece (3 hours)

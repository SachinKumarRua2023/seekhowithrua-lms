## API ENDPOINTS (Django Backend)

Base URL: https://api.seekhowithrua.com

Auth:
  POST /api/register/          → Create account (fields: email, password, confirm_password, role, first_name)
  POST /api/login/             → Login → returns token (field: email + password)
  POST /api/logout/            → Logout
  GET  /api/profile/           → Get profile (returns: { id, email, username, profile: { role } })
  PUT  /api/profile/update/    → Update profile

Voice Rooms (VCR) Panels:
  GET  /api/panels/                        → List all active panels
  POST /api/panels/create/                 → Create panel (fields: title, description, topic, max_members)
  POST /api/panels/<id>/join/              → Join panel → returns { host_peer_id }
  POST /api/panels/<id>/leave/             → Leave panel
  GET  /api/panels/<id>/members/           → Get members list
  POST /api/panels/<id>/raise-hand/        → Raise hand
  POST /api/panels/<id>/lower-hand/        → Lower hand
  POST /api/panels/<id>/mute-all/          → Mute all (host only)
  POST /api/panels/<id>/promote/<uid>/     → Promote to speaker
  POST /api/panels/<id>/kick/<uid>/        → Kick a member
  POST /api/panels/<id>/update-peer/       → Save peer_id to backend (fields: peer_id)

Future endpoints (not built yet):
  POST /api/vcr/follow/<user_id>/          → Follow a user
  DELETE /api/vcr/follow/<user_id>/        → Unfollow
  GET  /api/vcr/leaderboard/              → Rankings (?type=overall or ?type=college&college=X)
  POST /api/vcr/upvote/                    → Upvote a speaker (fields: to_user_id, panel_id)

Auth: Token-based (DRF). localStorage key: cosmos_token (for API), cosmos_user (for Navbar)
Master admin: email === "master@gmail.com" → isAdmin=true in VCR (Delete All button visible)
